Для того чтобы запустить проект - открываем terminal,
затем вставляем в терминал : git clone https://github.com/azamuzapbar/controlwork__9_Azat_Muzapbar

после клона, создаем виртуальное окружение, активируем и скачиваем зависимости
pip install -r requirments.txt

применяем миграции :
python manage.py makemigrations
python manage.py migrate

Запускаем проект:
python manage.py runserver
